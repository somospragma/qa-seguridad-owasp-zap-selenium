# Tecnologías

## Core Technologies
- **Java 17+**: Lenguaje de programación principal
- **Maven 3.6+**: Gestión de dependencias y build
- **Selenium WebDriver 4.15.0**: Automatización web
- **OWASP ZAP Client API**: Análisis de seguridad

## Testing Frameworks
- **JUnit 4**: Framework de testing
- **Allure 2.24.0**: Reportes avanzados
- **Cucumber**: BDD (Behavior Driven Development)

## Herramientas de Desarrollo
- **WebDriverManager**: Gestión automática de drivers
- **Chrome Browser**: Navegador para pruebas
- **IntelliJ IDEA / Eclipse**: IDEs recomendados

## Dependencias Maven
```xml
<dependency>
    <groupId>org.seleniumhq.selenium</groupId>
    <artifactId>selenium-java</artifactId>
    <version>4.15.0</version>
</dependency>
<dependency>
    <groupId>org.zaproxy</groupId>
    <artifactId>zap-clientapi</artifactId>
    <version>1.11.0</version>
</dependency>
```
