# Descarga

## Repositorio Principal
- **GitHub**: https://github.com/somospragma/qa-seguridad-owasp-zap-selenium
- **Rama Principal**: master

## Opciones de Descarga

### Clonar con Git
```bash
git clone https://github.com/somospragma/qa-seguridad-owasp-zap-selenium.git
```

### Descargar ZIP
1. Ir al repositorio en GitHub
2. Hacer clic en "Code" > "Download ZIP"
3. Extraer el archivo en el directorio deseado

### Fork del Proyecto
1. Hacer fork del repositorio en GitHub
2. Clonar tu fork:
```bash
git clone https://github.com/tu-usuario/qa-seguridad-owasp-zap-selenium.git
```
